## Cómo usar esta documentación

1. Lee [07-presentacion.md](./07-presentacion.md) primero — es la guía de entrega.
2. Completa los campos marcados con `<!-- TODO -->` o `[PENDIENTE]`.
3. Sube capturas a `assets/screenshots/` siguiendo la convención de nombres.
4. Graba y sube videos a `assets/videos/`.
5. Crea diagramas y guárdalos en `assets/diagrams/`.
