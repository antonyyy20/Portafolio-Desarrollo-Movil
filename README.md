## 👋 Presentación

¡Hola! Soy **Manuel Guillén**, estudiante de la materia de **Desarrollo Móvil**.

Este repositorio reúne el trabajo realizado a lo largo del semestre: desde ejercicios introductorios en **Kotlin** hasta aplicaciones Android completas con **Jetpack Compose**, **navegación**, **ViewModel**, **APIs del sistema** y un **proyecto final de gestión de tareas**.

El objetivo de este portafolio es documentar mi proceso de aprendizaje, demostrar la evolución de mis habilidades y servir como evidencia de las competencias adquiridas en el desarrollo de aplicaciones móviles para la plataforma Android.

> 💡 Cada carpeta del repositorio corresponde a una actividad numerada e incluye el código fuente, capturas, videos o documentación según el tipo de entrega.

## 📚 Descripción del curso

**Desarrollo Móvil** es una asignatura orientada a la construcción de aplicaciones nativas para dispositivos móviles, con énfasis en el ecosistema **Android** y el lenguaje **Kotlin**.

### Objetivos generales

| Área | Descripción |
|------|-------------|
| **Fundamentos** | Dominar la sintaxis de Kotlin, estructuras de control, funciones y programación orientada a objetos. |
| **Interfaz de usuario** | Diseñar interfaces con **XML/Views** y **Jetpack Compose**, aplicando principios de **Material Design 3**. |
| **Arquitectura** | Organizar el código con patrones como **MVVM**, separación de capas y buenas prácticas de Android. |
| **Navegación** | Implementar flujos entre pantallas con **Navigation Component** y **Compose Navigation**. |
| **Servicios del sistema** | Integrar capacidades nativas como **Text-to-Speech**, intents y comunicación con apps externas. |
| **Proyecto integrador** | Desarrollar una aplicación funcional de principio a fin con documentación técnica. |

### Metodología de aprendizaje

El curso combina **teoría**, **talleres prácticos**, **tareas individuales**, **evaluaciones parciales** e **investigación aplicada**. Las actividades van incrementando en complejidad: se parte de programas de consola y se avanza hacia apps con estado, persistencia, navegación y servicios del dispositivo.

## 🎓 Conclusión del curso

Completar la materia de **Desarrollo Móvil** representó un recorrido significativo: pasé de escribir programas de consola en Kotlin a construir aplicaciones Android funcionales con arquitectura definida, navegación entre pantallas e integración con servicios nativos del dispositivo.

Los talleres prácticos —como el de **LazyColumn**, el juego de **adivinar número** y la integración con **WhatsApp**— me permitieron experimentar con distintos enfoques de UI (XML vs Compose) y comprender cuándo aplicar cada uno. La **tarea de navegación** consolidó el manejo de flujos complejos, mientras que el módulo de **Text-to-Speech** me acercó a las capacidades hardware/software del teléfono.

Las evaluaciones (**Quiz**, **Ahorcado** y **Parcial #2**) pusieron a prueba la capacidad de implementar lógica de negocio bajo restricciones de tiempo, y la **investigación de System Design** amplió la visión más allá del código, considerando escalabilidad y buenas prácticas de arquitectura.

El **proyecto final de gestión de tareas** fue la síntesis de todo lo aprendido: diseño, implementación, documentación y demostración de una app completa lista para ser presentada.

En conclusión, este curso me proporcionó una base sólida para continuar desarrollando aplicaciones móviles profesionales. Las habilidades adquiridas —desde Kotlin hasta Jetpack Compose, pasando por MVVM y servicios del sistema— son directamente aplicables en entornos de desarrollo real. Quedo motivado para profundizar en temas como persistencia avanzada, testing, publicación en Play Store y arquitecturas más robustas como Clean Architecture.
